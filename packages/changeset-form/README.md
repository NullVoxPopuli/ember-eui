@ember-eui/changeset-form
==============================================================================

This package depends on [`@ember-eui/core`](../core/README.md), so follow up the installation instructions over there.

Compatibility
------------------------------------------------------------------------------

* Ember.js v3.24 or above
* Ember CLI v3.24 or above
* Node.js v12 or above
